# 动画

### 使用

1.安装

`npm i react-transition-group -S`

2.引入

```jsx
import {CSSTransition} from 'react-transition-group'
```

3.参数

* in={this.state.show} 触发条件
* timeout={1000} 过渡时间
* classNames='var'

```
var-enter：入场动画之前
var-enter-active：入场动画执行时
var-enter-done：入场动画执行完
var-exit：出场动画之前
var-exittive：出场
var-exitne：出场动画执行完
```

* unmountOnExit 移除dom

钩子

* onEntered:入场动画执行完的钩子
